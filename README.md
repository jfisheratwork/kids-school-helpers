# Kids School Helpers

A collection of lightweight, client-side web applications designed as educational tools.

Currently Hosted Applications:
- **Pre-Algebra Solver**: An interactive tool for generating worksheets and solving multi-step linear equations.

## Development Setup

If you wish to run the project locally or contribute, please see our [Developer Onboarding Guide](developer.md).

## Architecture

This project is built as a series of Single Page Applications (SPAs) using Vanilla JavaScript, HTML5, and Tailwind CSS. We emphasize zero-build-step development where possible, utilizing CDNs and static asset hosting via GitHub Pages.

Please refer to `Agents.md` for our automated contributor guidelines.
