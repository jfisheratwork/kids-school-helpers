# 03 - Worksheet Generator Mode

## Overview
The Worksheet Generator Mode allows users to generate printable pre-algebra worksheets with multi-step linear equations.

## Layout and Formatting
* **Layout:** Responsive grid system for screen viewing; specific CSS `@media print` rules for standardized 8.5x11 inch physical output.
* **Formatting:** Equations must utilize the Mathematical Rendering Engine. Sufficient whitespace must be allocated per problem for manual computation.

## Features
* **Toggle:** UI switch to toggle visibility of the answer key prior to printing.
