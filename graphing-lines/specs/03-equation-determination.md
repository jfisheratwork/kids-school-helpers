# 03 - Equation Determination Mode

## Overview
The reverse of the Interactive Graphing mode. The application provides a pre-drawn line on the coordinate plane, and the student must determine the correct algebraic equation.

## State Machine
1. **Identify the Y-Intercept:**
   - The user must click the exact point where the line crosses the y-axis and input the $b$ value into a text field.
2. **Determine the Slope:**
   - The user must select any other valid integer coordinate on the line, and input the "rise" and "run" fractions to calculate the slope $m$.
3. **Formulate the Equation:**
   - The user pieces together the final $y = mx + b$ equation.

## Hint System
* **Y-Intercept Hint:** An animation flashes the y-axis intersection.
* **Slope Hint:** An animation highlights two clear integer coordinates on the line and draws the "rise over run" triangle connecting them.
