# 04 - Worksheet Generator Mode

## Overview
Allows teachers/parents to generate printable worksheets combining equations and graphing paper.

## Features
* **Grid Formatting:** Procedurally generates perfectly square SVG coordinate grids explicitly scaled for standard 8.5x11 inch physical print output via `@media print`.
* **Problem Types:**
  * **Graph the Equation:** Prints an equation with a blank coordinate grid next to it.
  * **Find the Equation:** Prints a line on a coordinate grid with blank spaces for the student to write the equation.
* **Answer Key:** A toggle switch that overlays the correct lines (in red/blue) onto the blank grids or prints the correct equations next to the graphs for grading purposes.
